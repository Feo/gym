photos/:id

获取照片

URL

http://localhost:3000/api/v1/photos/:id

支持格式

JSON

HTTP请求方式

GET

是否需要登录

是

请求参数

无

注意事项

无

返回结果

JSON示例

                    [
                        {
                            "created_at": "2014-10-06T17:02:53+08:00",
                            "id": 9,
                            "image": {
                                "url": "/uploads/photo/image/9/photo.jpeg"
                            },
                            "image_file_name": null,
                            "image_file_size": 0,
                            "member_id": 1,
                            "time": "20141006",
                            "title": "33",
                            "updated_at": "2014-10-06T17:02:53+08:00"
                        },
                        "/uploads/photo/image/9/photo.jpeg"
                    ]

返回字段说明

返回值字段 字段类型 字段说明
created_at datetime 数据创建时间
id integer 图片ID
image hash 图片
image_file_name string 图片文件名
image_file_size integer 图片文件尺寸
member_id integer 会员ID
time string 上传时间
title string 图片名称
updated_at datetime 数据更新时间

其他

无
