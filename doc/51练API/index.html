Coaches
读取接口 api/v1/coaches/info 获取当前教练信息
api/v1/coaches/:id/info 获取指定教练信息
api/v1/coaches/apply_list 获取请求关联会员列表
api/v1/coaches/related_list 获取已关联会员列表
写入接口 api/v1/coaches/register 创建新教练账户
api/v1/coaches/login 教练登陆
api/v1/coaches/logout 教练退出
api/v1/coaches/update 新建/更新当前教练信息
api/v1/coaches/change_pwd 修改教练密码
api/v1/coaches/approve_apply 批准会员关联请求
api/v1/coaches/refuse_apply 拒绝会员关联请求
api/v1/coaches/delete_member 取消会员关联
api/v1/coaches/send_token 发送激活码
api/v1/coaches/activate_account 激活账户
api/v1/coaches/reset_password 重置密码
api/v1/coaches/find_member 查找会员
api/v1/coaches/apply_member 申请关联会员

api/v1/members/apply_coach 申请关联教练
api/v1/members/delete_coach 取消教练关联

MemberHabits

api/v1/memberhabits/update 修改用户健康习惯表

Events
写入接口 api/v1/events/coach_create 教练创建日程
api/v1/events/member_create 会员创建日程
api/v1/events/member_approve 会员同意日程
api/v1/events/coach_approve 教练同意日程
api/v1/events/coach_update 教练更新日程
api/v1/events/member_update 会员更新日程
api/v1/events/coach_delete 教练删除日程
api/v1/events/member_delete 会员删除日程
api/v1/events/member_refuse 会员拒绝日程
api/v1/events/coach_refuse 教练拒绝日程
api/v1/events/member_waiting_events 获取等待会员同意日程表
api/v1/events/member_approved_events 获取会员已同意日程表
api/v1/events/member_all_events 获取会员所有日程表
api/v1/events/coach_waiting_events 获取教练已同意日程表
api/v1/events/coach_approved_events 获取教练已同意日程表
api/v1/events/coach_all_events 获取教练所有日程表
api/v1/events/:id 获取日程
Messages
读取接口 api/v1/messages/coach_message 获取当前教练所有消息
api/v1/messages/member_message 获取当前会员所有消息
api/v1/messages/:id 获取某条消息
写入接口 api/v1/messages/create 创建新消息
api/v1/messages/update 更新消息
api/v1/messages/coach_delete 教练删除消息
api/v1/messages/member_delete 会员删除消息
api/v1/messages/coach_member_messages 教练获取与某个会员的所有消息
api/v1/messages/member_coach_messages 会员获取与某个教练的所有消息
api/v1/messages/coach_delete_all 教练删除与某个会员的所有消息
api/v1/messages/member_delete_all 会员删除与某个教练的所有消息
