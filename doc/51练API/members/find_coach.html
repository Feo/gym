members/find_coach

会员查找教练

URL

http://localhost:3000/api/v1/members/find_coach

支持格式

JSON

HTTP请求方式

POST

是否需要登录

是

请求参数

  必选 类型及范围 说明
[coach][nickname] true string 昵称
[coach][name] true string 姓名
[coach][province] true string 省
[coach][city] true string 市
[coach][district] true string 区
[coach][street] true string 街道
[coach][profession] true string 专业方向
[coach][gender] true string 性别
[coach][email] true string 邮箱
[coach][organization] true string 所属机构
[coach][age_less] true integer 年龄小于
[coach][age_greater] true integer 年龄大于
[coach][experience_less] true integer 工作经验小于
[coach][experience_greater] true integer 工作经验大于
[coach][grade_less] true integer 评分小于
[coach][grade_greater] true integer 评分大于

注意事项

无
