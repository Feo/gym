assessments/all

获取某个会员的所有评估

URL

http://localhost:3000/api/v1/assessments/all

支持格式

JSON

HTTP请求方式

POST

是否需要登录

是

请求参数

  必选 类型及范围 说明
[member_id] true integer 会员ID

注意事项

无
