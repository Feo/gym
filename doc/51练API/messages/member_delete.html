messages/member_delete

会员删除消息

URL

http://localhost:3000/api/v1/messages/member_delete

支持格式

JSON

HTTP请求方式

POST

是否需要登录

是

请求参数

  必选 类型及范围 说明
[id] true string 消息ID

注意事项

无

返回结果

JSON示例

                    {
                        "coach_phone": "",
                        "coach_phone_array": "18600000001;",
                        "content": "test",
                        "created_at": "2014-10-06T21:39:22+08:00",
                        "id": 43,
                        "member_phone": "18664913341;18664913340;18000000001;",
                        "member_phone_array": "18664913341;18664913340;18000000001;",
                        "updated_at": "2014-10-06T21:41:43+08:00"
                    }

返回字段说明

返回值字段 字段类型 字段说明
coach_phone string 教练手机号
coach_phone_array string 教练手机号(保留字段)
content string 消息内容
created_at datetime 创建时间
id integer 消息ID
member_phone string 会员手机号
member_phone_array string 会员手机号(保留字段)
updated_at datetime 数据更新时间

其他

无
