events/coach_all_events

获取教练所有日程表

URL

http://localhost:3000/api/v1/events/coach_all_events

支持格式

JSON

HTTP请求方式

POST

是否需要登录

是

请求参数

  必选 类型及范围 说明
[begin_date] true string 开始日期
[end_date] true string 结束日期
[page] true integer 页数
[per_page] true integer 每页日程数

注意事项

无

返回字段说明

返回值字段 字段类型 字段说明
begin_date string 起始日期
coach_approved boolean 教练是否同意
coach_id integer 教练ID
content string 日程内容
created_at datetime 创建时间
date string 具体日期
end_date string 结束日期
id integer 日程ID
member_approved string 已同意该日程会员号码
member_phone string 会员号码
time string 具体时间
updated_at datetime 数据更新时间
week string 周期性星期
